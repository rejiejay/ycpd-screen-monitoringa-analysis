# 养车频道数据监控平台

## 项目搭建

``` bash
# 初始化项目
npm install

# 运行开发环境, 服务端端口热加载 at localhost:8000
npm run dev

# 项目打包 到 测试环境
npm run builddev

# 项目打包 到 生产环境
npm run build

```
